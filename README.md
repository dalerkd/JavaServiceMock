# JavaServiceMock
为前端提供了一种便捷的json,mock方式.




## 添加一项mock:
- x-www-form-urlencoded,post方式地址: http://127.0.0.1:8080/mocks
- mockPath：    "/work"   ;mock了work路径
- mockMethod：  "GET"   ;mock的方法为GET方式
- returnJson：   {"Hi":"This a test"}  ;mock返回结果
- returnStatus： 400     ;mock返回结果
